Şifresiz sudo erişimi

    $ su

Bu komuttan sonra root şifreniz istenecek ve makinenizde root olacaksınız

    $ visudo

Bu sudo nun dosya ayarı, sadece root iken giriş yapabileceğiniz bir ayar dosyası

  • Bazı kemik sistemlerde sudo sisteminize kurulu olmadan yüklenir eğer sisteminizde sudo kurulu değilse root olup

    $ aptitude install sudo

    ile kurabilirsiniz

Presenter Notes

Source: /home/aydintd/github/fo/vagrant-2/index.md
11/18

Visudo

Bu dosya sudo izinlerinin kime verileceğini tanımlayabildiğiniz bir ayar dosyası Aşağıdaki satırları dosyaya sırasıyla ekleyin

    Defaults env_keep="SSH_AUTH_SOCK"

  • Visudo dosyasında sıra önemli o yüzden diğer değişkenlerin bulunduğu kısmın hemen altına env_keep değişkenini ekleyin

  • Şimdi admin grubuna şifresiz erişim hakkını vermemiz gerekiyor

    %admin ALL=NOPASSWD: ALL

    satırını %sudo satırının hemen altına ekleyin ve %sudo satırını commentleyin.

12/18

Visudo

Dosyaya admin grubunu ekledik ama sistemimizde admin diye bir grup yok Şimdi bu grubu oluşturalım ve kullanıcımızın gruplarına ekleyelim

    $ groupadd admin
    $ usermod -aG admin vagrant

  • Tüm bu ayarların aktive olması için sistemimizi yeniden başlatmamız gerekiyor

  • Sistem tekrar açıldığında

    $ sudo which sudo
> /usr/bin/sudo

    gibi bir çıktıyı şifre girmeden görmemiz gerekiyor

  • Buraya kadar her şey normalse Virtual Box Misafir Eklentileri kurulumuna geçebiliriz

VirtualBox Guest Additions

Vagrant a tüm gücü şifresiz erişim hakkıyla verdik

Şimdi yapmamız gereken paylaşılan dosyalar ve port yönlendirmeleri için VirtualBox un bize sunduğu misafir eklentilerini kurmamız gerekecek

Aşağıdaki işlemleri sırasıyla yapalım

    $ sudo apt-get install linux-headers-$(uname -r) build-essential

  • Misafir eklentileri imajını VirtualBox penceresindeki Araçlar kısmında Install Guest Additions a tıklayarak yerleştirelim ve

    $ sudo mount /dev/cdrom /media/cdrom
$ sudo sh /media/cdrom/VBoxLinuxAdditions.run

    diyerek misafir eklentilerini kuralım

  • Eğer ortamı GUI olmadan kurmuşsanız Guest Additions kurulumun sonunda Pencere Sistem Sürücüleri veya OpenGL eksikliği hakkında uyarı verecektir bu uyarıları görmezden gelebilirsiniz

SSH Konfigürasyonu

  • SSH Sunucusunu sistemimize kuralım

    $ sudo apt-get install ssh
$ echo 'UseDNS no' >>/etc/ssh/sshd_config

  • /home dizinimiz içinde .ssh dizinini oluşturup Vagrantın bize sağladığı ssh anahtarını authorized_keys dosyası içine kopyalamamız gerekiyor

    $ mkdir .ssh
$ wget --no-check-certificate -0 .ssh/authorized_keys
https://raw.github.com/mitcellh/vagrant/master/keys/vagrant.pub
$ chmod 700 .ssh
$ chmod 600 .ssh/authorized_keys

$ sudo apt-get clean
$ sudo poweroff

  • Bütün ayarlar bitti artık imajımızın son halini paketleyip kutu haline getirebiliriz

16/18

Kutu Hazırlama

  • Imajı kutu haline getirmek için yapmamız gereken tek şey .vbox uzantılı dosyayla aynı dizin içersinde aşağıdaki komutu yazmak

    $ vagrant package --base kutu_adi

  • Kutunun hazırlanma süreci bir kaç dakika kadar sürebilir, işlem bittiğinde bulunduğunuz dizin içinde package.box u göreceksiniz

  • Kutuya bağlanmak için aşağıdaki komutları sırasıyla verin

    $ vagrant box add kutu_adi package.box
$ mkdir test_ortami
$ cd test_ortami
$ vagrant init kutu_adi
$ vagrant up
$ vagrant ssh

Eğer buraya kadar her şeyi doğru yapmışsanız hazırladığınız imaja Vagrantla ulaşabileceksiniz
